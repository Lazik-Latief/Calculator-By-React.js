# React Calculator

This is a simple and clean calculator app built using **React** and **Vite**.  
I created this project to practice React basics, state management, and component structure.

---

##  Features
- Basic arithmetic operations
- Responsive UI
- Clear and easy-to-use layout
- Fast performance using Vite

---

##  Tech Stack
- **React**
- **JavaScript**
- **Vite**
- **CSS**

---

##  What I Learned
- Using `useState` for managing values
- Handling button events
- Structuring components
- Making a small project using React
